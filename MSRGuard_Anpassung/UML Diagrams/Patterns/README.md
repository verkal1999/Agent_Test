# UML Diagrams / Patterns

This folder is part of the `MA_Python_Agent` master's thesis repository. It stores UML diagrams that explain recurring design patterns used in the adapted MSRGuard runtime.

## What this folder contains

- `Command.plantuml`: command pattern as used by command-force execution
- `Factory.plantuml`: factory pattern used for force creation
- `Observer.plantuml`: observer pattern used for runtime notifications
- `Specification.plantuml`: specification-style selection logic
- `Strategy.plantuml`: strategy pattern for interchangeable behavior

## Current status

These diagrams are explanatory thesis documentation. They help communicate design choices made in the runtime implementation.
