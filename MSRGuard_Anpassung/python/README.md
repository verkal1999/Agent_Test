# python

This folder is part of the `MA_Python_Agent` master's thesis repository. It contains the Python side of the adapted MSRGuard setup.

## What this folder contains

- `msrguard/`: the Python package with both agent implementations, evaluation helpers, and D2 analysis logic
- `agent_results/`: raw artifacts generated by Streamlit sessions and agent runs

## Current status

If you are looking for the active thesis logic, start with `msrguard/`. The `agent_results/` folder stores outputs rather than source code.
