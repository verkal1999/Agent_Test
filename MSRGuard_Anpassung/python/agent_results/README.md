# agent_results

This folder is part of the `MA_Python_Agent` master's thesis repository. It stores raw runtime artifacts generated by Streamlit sessions and agent executions.

## What this folder contains

- `*_event.json`: input event payloads used by an agent run
- `*_result.json`: result JSON written by an agent run when available
- `streamlit_<timestamp>_<correlation>/`: per-session folders with chat transcripts and UI-related run artifacts

## Important distinction

This folder stores raw operational artifacts.

The curated benchmark outputs used in the thesis comparison are stored separately in:

- `Evaluation/results/`

## Current status

The folder can grow quickly during experiments. It is useful for debugging and traceability, but it is not the authoritative source for the summarized evaluation tables in the thesis.
