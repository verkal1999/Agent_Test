# Generierte_PLCOpenXML

This folder is part of the `MA_Python_Agent` master's thesis repository. It stores generated PLCopen XML samples used in notebook-based experiments.

## What this folder contains

- `FB_GeneratedJob_01.xml`: example generated PLCopen XML for a function block

## Current status

The folder is mainly a small thesis-side sandbox for generated XML artifacts, not the main source location for runtime PLC exports.
