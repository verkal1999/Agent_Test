# PLC2Skill_Dateien

This folder is part of the `MA_Python_Agent` master's thesis repository. It stores the external PLC2Skill command-line binary used in supporting experiments.

## What this folder contains

- `plc2skill-cli-2.1.1.jar`: PLC2Skill CLI packaged as a Java archive

## Notes

- The file is an external tool, not a Python module from this repository.
- Typical usage is via `java -jar plc2skill-cli-2.1.1.jar`.
- The folder exists mainly to keep the thesis workspace self-contained.
