# Pipelines

This folder is part of the `MA_Python_Agent` master's thesis repository. It groups the data-preparation pipelines that convert engineering artifacts into machine-readable diagnosis inputs.

## What this folder contains

- `IngestionPipeline/`: the main thesis pipeline for building and enriching knowledge-graph artifacts from TwinCAT and PLCopen sources

## Current status

At the moment the repository contains a single active pipeline family in this folder. The folder is intentionally small because most logic is implemented inside `Pipelines/IngestionPipeline/`.
